# HazeOver — Portable Focus Mode That Dims Background Windows

## About HazeOver

HazeOver provides a complete focus environment for users who want to eliminate distractions from background windows. HazeOver combines window dimming, adjustable intensity, smooth transitions, and multi‑display support. HazeOver enables users to darken all inactive windows, keeping only the active window bright and focused. HazeOver runs on desktop platforms. HazeOver is trusted by writers, developers, designers, students, and productivity enthusiasts worldwide.

## Deep Dive — Focus Mode Notes

Window dimming: dim all inactive windows, adjustable intensity (0‑100%)
Smooth transitions: fade in/out, dim on app switch, animation speed
Multi‑display: dim all displays, independent settings per display
Exceptions: whitelist apps, exclude certain apps from dimming
Hotkeys: toggle on/off, increase/decrease dim, temporary disable

## Quick Reference for HazeOver

| Operation | HazeOver Action |
|-----------|-----------------|
| Toggle HazeOver on/off | Cmd+Shift+H |
| Increase dim intensity | Cmd+Shift++ |
| Decrease dim intensity | Cmd+Shift+- |
| Reset dim intensity | Cmd+Shift+0 |
| Temporary disable (hold) | Hold Shift |
| Open preferences | Cmd+, |
| Show help | Cmd+Shift+/ |
| Quit HazeOver | Cmd+Q |
